üß† Core Concepts

1. CSS Keyframe Animations

CSS animations allow elements to transition between styles smoothly.

Key Properties:

  • @keyframes - Defines the animation sequence
  • animation - Applies the keyframe animation
  • animation-duration - How long the animation takes
  • animation-iteration-count - Number of repetitions (infinite for loops)
  • animation-timing-function - Easing (ease, linear, ease-in-out)

2. SVG (Scalable Vector Graphics)

SVGs are resolution-independent graphics defined in XML format.

Advantages:

  • Scalable without quality loss
  • Manipulable via JavaScript
  • Styleable with CSS
  • Small file sizes for simple graphics

3. GSAP (GreenSock Animation Platform)

GSAP is a powerful JavaScript animation library.

Key Methods:

  • gsap.to() - Animate to specific values
  • gsap.from() - Animate from specific values
  • gsap.to(element, {duration, property, ease})
  • Easing functions: back.out, elastic.out, sine.inOut, power2.out

4. GSAP Draggable

Enables drag-and-drop functionality with physics-based interactions.

Key Features:

  • Inertia (momentum-based movement)
  • Bounds constraints
  • Callbacks (onDragStart, onDrag, onDragEnd)

Step 4: GSAP Animations

A. Entrance Animation

JavaScript 
// Animate element appearing from above
gsap.from(element, {
    y: -200,           // Start 200px above
    opacity: 0,        // Start invisible
    scale: 0.5,        // Start at half size
    duration: 0.8,     // Animation duration
    delay: 0.2,        // Delay before starting
    ease: 'back.out(1.7)'  // Easing function
});

B. Continuous Swaying Animation

JavaScript 
function startSwaying(element) {
    const swayAmount = 5;  // Degrees of rotation
    const duration = 2;    // Seconds per sway
    
    gsap.to(element, {
        rotation: swayAmount,
        duration: duration,
        ease: 'sine.inOut',
        repeat: -1,        // Infinite repeat
        yoyo: true         // Reverse animation
    });
}

Step 5: Drag and Drop with GSAP Draggable

JavaScript 
// Make element draggable
Draggable.create(element, {
    type: 'x,y',           // Allow horizontal and vertical movement
    bounds: '.container',  // Constrain to container
    inertia: true,         // Physics-based momentum
    
    onDragStart: function() {
        // Actions when drag starts
        element.classList.add('dragging');
        gsap.to(element, { scale: 1.1, duration: 0.3 });
    },
    
    onDrag: function() {
        // Actions during drag
        console.log('Dragging at:', this.x, this.y);
    },
    
    onDragEnd: function() {
        // Actions when drag ends
        element.classList.remove('dragging');
        gsap.to(element, { 
            scale: 1, 
            duration: 0.5, 
            ease: 'elastic.out(1, 0.5)' 
        });
    }
});

function createSparkles(x, y) {
    const container = document.getElementById('elementsContainer');
    
    for (let i = 0; i < 5; i++) {
        // Create sparkle element
        const sparkle = document.createElement('div');
        sparkle.className = 'sparkle';

function windEffect() {

Color Utility Function

JavaScript 
// Darken or lighten a hex color
function shadeColor(color, percent) {
    const num = parseInt(color.replace('#', ''), 16);
    const amt = Math.round(2.55 * percent);
    const R = (num >> 16) + amt;
    const G = (num >> 8 & 0x00FF) + amt;
    const B = (num & 0x0000FF) + amt;
    
    return '#' + (0x1000000 + 
        (R < 255 ? R < 1 ? 0 : R : 255) * 0x10000 + 
        (G < 255 ? G < 1 ? 0 : G : 255) * 0x100 + 
        (B < 255 ? B < 1 ? 0 : B : 255)
    ).toString(16).slice(1);
}

// Usage
const darkerColor = shadeColor('#FF69B4', -20);  // Darker
const lighterColor = shadeColor('#FF69B4', 20);  // Lighter

üí° Tips & Best Practices

Performance

  1. Use transform and opacity - These properties are GPU-accelerated
  2. Avoid animating width, height, margin - Causes layout reflows
  3. Use will-change sparingly - Only on elements that will animate
  4. Clean up elements - Remove elements after animations complete
CSS 
.animated-element {
    will-change: transform, opacity;
    transform: translateZ(0);  /* Force GPU acceleration */
}

Accessibility

Respect prefers-reduced-motion for users who prefer less animation:

CSS 
@media (prefers-reduced-motion: reduce) {
    * {
        animation-duration: 0.01ms !important;
        animation-iteration-count: 1 !important;
    }
}

Browser Compatibility

  • GSAP works in all modern browsers
  • CSS animations have excellent support
  • Test on mobile devices for touch interactions
